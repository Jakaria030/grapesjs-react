# GrapesJS Project

A simple web page builder built with React, Vite, and **GrapesJS**. It allows users to visually build web pages using drag-and-drop components.


## 🚀 Tech Stack

- React
- GrapesJS
- HTML5
- CSS3
- Vite

## 📦 Getting Started
Follow these steps to set up and run the project locally:

1. Clone the repository:
   ```bash
   git clone git@github.com:Jakaria030/grapesjs-react.git
   ```
2. Navigate to the project directory:
   ```bash
   cd grapesjs-react
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Run the development server:
   ```bash
   npm run dev
   ```
